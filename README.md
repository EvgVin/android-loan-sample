This is a sample project of a loan application for Android
==========================================================

## Technical Specification

* Kotlin
* MVVM
* Dagger 2
* DataBinding 2
* RxJava 2
* Retrofit 2
* Navigation
* Places
* CameraView
* Glide
* ConstraintLayout
* Material

The project is under development and some features have not yet been implemented.
